There are following steps we will be taking to solve the activity selection problem using Greedy method in Java,

  • Sort the activities in ascending order according to their finishing time.
  • Select the first activity from sorted array a[] (Whatever you assume) and reupdate it.
  • If the start time of the currently selected activity is greater than or equal to the finish time of the previously selected activity, then print the current activity.
  • Select the next activity in a[] array.
  • Repeat this activity till the last index.
