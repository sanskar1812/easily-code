Checking if two trees are mirror in Java

What are mirror trees?

Given 2 trees, if mirror representation of one tree exactly represents the other then those trees are mirror trees. In other words, the left subtree of one tree should exactly match with right subtree of other tree and vice-versa.

Algorithm

  • We first check if both trees are non-empty.
  • If yes, we return true.
  • Else we check if either one is non-empty, in that case, we return false.
  • Since we need the left subtree of one tree to be equal to right subtree of other tree and vice-versa.
  • We recursively call our function by passing left subtree of one tree and right subtree of another tree.
  • Again we call the function by passing right subtree of one tree and left subtree of the other tree.
